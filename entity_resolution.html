Methodology

A description of the methodology is here.

Briefly, there are 4 cases:

  • Case 1: Match. Algorithm used direct comparison between entities and exceeds a threshold.
  • Case 2: Match. Inputs match the same key.
  • Case 3: Match. Inputs match a different keys, but keys match same company.
  • Case 4: Non-Match. Inputs match a different keys and companies.

Examples:

  • Case 2: "Facebook" vs "Facebooc" (misspelling)
  • Case 3: "Facebook" vs "Meta Platforms" (different company name)
  • Case 3: "Bank of America" vs "BofA" (different expression of company name)
  • Case 4: "Facebook" vs "Apple"
